## 内部版本号 {#build-no-javascript-sdk-410}

Adobe Pass身份验证： JavaScript 4.1.0

发行日期： **10/04/2018**


## 发行版概述 {#overview-javascript-sdk-410}

* Access Enabler JS v4.1.0现在遵循Safari的第三方Cookie策略和智能防跟踪机制。 现在，所有身份验证流在所有Safari浏览器版本中都可正常工作。
