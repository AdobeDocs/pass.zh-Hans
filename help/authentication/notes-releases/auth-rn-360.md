### 发行版概述 {#release-overview-360}

此版本重点关注Adobe Pass身份验证服务的质量、稳定性和安全性更新。

#### 增强功能

* 对REST API V2的可靠性、可观察性和响应一致性进行了常规改进。
* 更广泛地强化身份验证流程和请求处理。
* 对合作伙伴和MVPD集成行为进行了改进，以提高常见方案中的兼容性。
